# 公司官网项目

## 项目概述
这是一个现代化的响应式公司官网项目，采用HTML5和CSS3技术开发。

### 项目结构
```
company-website/
├── index.html              # 首页
├── about/                  # 关于我们
│   └── index.html
├── products/              # 产品服务
│   └── index.html
├── news/                 # 新闻中心
│   └── index.html
├── contact/             # 联系我们
│   └── index.html
├── careers/            # 招聘信息
│   └── index.html
├── assets/            # 静态资源
│   ├── css/          # 样式文件
│   ├── js/           # JavaScript文件
│   ├── images/       # 图片资源
│   └── fonts/        # 字体文件
└── locales/          # 多语言文件
    ├── zh/           # 中文
    └── en/           # 英文
```

### 技术栈
- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript
- 响应式设计

### 主要功能
- 响应式布局，适配多端设备
- 多语言支持（中英文）
- SEO优化
- 社交媒体集成
- 用户反馈表单

### 页面说明
1. 首页
   - 公司简介展示
   - 产品服务概览
   - 最新动态
   - 快速联系入口

2. 关于我们
   - 公司历史
   - 企业文化
   - 团队介绍

3. 产品服务
   - 产品分类展示
   - 详细介绍页面

4. 新闻中心
   - 公司新闻
   - 行业动态

5. 联系我们
   - 联系表单
   - 公司地址信息
   - 联系方式

6. 招聘信息
   - 职位列表
   - 招聘流程

### 开发规范
- 使用语义化HTML标签
- CSS采用BEM命名规范
- 代码添加中文注释

- 图片资源优化
